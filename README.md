# Proyecto Turismo - La Rioja

Trabajo Práctico N° 3 de la materia Programación Web — TUP, FRLR.

## Descripción

Sitio web de 4 páginas que muestra el uso de Flexbox, JavaScript común (manipulación del DOM, eventos, arreglos) y React, utilizando como temática lugares turísticos de La Rioja, Argentina.

## Páginas

- **index.html**: portada con navegación hacia los 3 ejercicios.
- **flexbox.html**: tarjetas de lugares turísticos generadas dinámicamente con JavaScript, organizadas con Flexbox. Incluye filtro por categoría y ordenamiento alfabético.
- **personas.html**: formulario para registrar personas (nombre, apellido, edad, altura, peso), con tabla dinámica, cálculo de IMC y eliminación de filas. Resuelto con JavaScript común.
- **react.html**: misma funcionalidad que personas.html, resuelta con componentes y estado de React (vía CDN, sin proceso de build).

## Tecnologías utilizadas

- HTML5
- CSS3 (Flexbox)
- JavaScript (DOM, eventos, arreglos)
- React 18 (CDN + Babel Standalone)
